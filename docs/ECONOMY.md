# SUIBOX 经济模型设计文档

## 📌 版本
v1.0 | 2026-03-18

---

## 1️⃣ 核心思路

| 阶段 | 目标 | 人性设计 |
|------|------|---------|
| 邀请 | 获得用户 | 轻松上手 → 首次体验 |
| 开盲盒 | 获得碎片 | 每次都有收获 → 期待感 |
| 合成 | 获得NFT | 努努力就够得着 → 成就感 |
| 升级 | 高级NFT | 有难度但不是不可能 → 追求感 |
| 交易 | 流通 | 差价获利 → 交易动力 |

---

## 2️⃣ 玩家分层与动机

| 玩家类型 | 目标 | 行为 | 贡献 |
|---------|------|------|------|
| 免费玩家 | 白嫖体验 | 开盲盒、合成普通 | 活跃度、流量 |
| 微氪玩家 | 省钱省时间 | 买碎片、快速合成 | 消费碎片 |
| 中氪玩家 | 获得稀有 | 买NFT、升级 | 消费NFT |
| 工作室 | 批量刷 | 脚本开盒、养号 | 流动性 |
| 商人 | 赚差价 | 低买高卖 | 市场深度 |

## 3️⃣ 完整数据模型

### 3.1 邀请阶段

| 任务 | 被邀请人奖励 | 邀请人奖励 |
|------|-------------|-----------|
| 1位好友开盒 | +1次开盒 | +0.5 BOX |
| 3位好友开盒 | +2次开盒 | +1.5 BOX |
| 10位好友开盒 | +3次开盒 | +5 BOX |

### 3.2 开盲盒阶段

| 结果 | 概率 | 获得碎片 |
|------|------|---------|
| 普通碎片 | 50% | 1-2个 |
| 稀有碎片 | 15% | 1个 |
| 史诗碎片 | 3% | 1个 |
| 感谢参与 | 32% | 0（但计入保底） |

### 3.3 保底机制

| 保底次数 | 必得 |
|---------|------|
| 3次感谢 | 1个普通碎片 |
| 7次感谢 | 1个稀有碎片 |
| 25次感谢 | 1个史诗碎片 |

### 3.4 碎片合成

| 消耗 | 产出 | 奖励BOX | 难度 |
|------|------|---------|------|
| 6普通碎片 | 1普通NFT | +5 BOX | ⭐ 容易 |
| 8稀有碎片 | 1稀有NFT | +8 BOX | ⭐⭐ 中等 |
| 10史诗碎片 | 1史诗NFT | +15 BOX | ⭐⭐⭐ 难 |

### 3.5 NFT升级

| 消耗 | 产出 | 消耗BOX | 奖励BOX | 净消耗 |
|------|------|---------|---------|--------|
| 5普通NFT + 30 BOX | 1稀有NFT | -30 | +10 | -20 BOX |
| 4稀有NFT + 50 BOX | 1史诗NFT | -50 | +20 | -30 BOX |

### 3.6 Staking（持有产出）

| 持有NFT | 每日产出 |
|---------|---------|
| 普通NFT | +0.1 BOX/天 |
| 稀有NFT | +0.5 BOX/天 |
| 史诗NFT | +2 BOX/天 |

### 3.7 NFT发行量

| 类型 | 发行量 | 定位 |
|------|--------|------|
| 普通NFT | 100万 | 普及 |
| 稀有NFT | 10万 | 稀缺 |
| 史诗NFT | 2万 | 稀有 |

## 4️⃣ 经济循环

```
交易所买BOX → 开盲盒/合成 → NFT升级（销毁BOX）→ Staking产出BOX
                                    ↓
                              BOX通缩 → 价格上涨 → 更多人买
```

| 行为 | 净BOX变化 |
|------|----------|
| 碎片合成产出 | +BOX（吸引玩家） |
| NFT升级销毁 | -BOX（通缩） |
| Staking | +BOX（持有收益） |

## 5️⃣ 模拟数据（100万人开盒）

### 5.1 碎片产出
| 碎片 | 概率 | 数量 |
|------|------|------|
| 普通 | 50% | 50万 |
| 稀有 | 15% | 15万 |
| 史诗 | 3% | 3万 |

### 5.2 合成NFT
| 类型 | 碎片 | 可合成NFT |
|------|------|----------|
| 普通 | 50万÷6 | 8.3万 |
| 稀有 | 15万÷8 | 1.9万 |
| 史诗 | 3万÷10 | 0.3万 |

### 5.3 升级NFT
| 升级 | 消耗NFT | 消耗BOX | 产出NFT |
|------|---------|---------|--------|
| →稀有 | 8.3万×5=41.5万 | 41.5万×30=1245万 | 8.3万稀有NFT |
| →史诗 | 3.6万×4=14.4万 | 14.4万×50=720万 | 0.36万史诗NFT |

### 5.4 经济
| 阶段 | 净BOX变化 |
|------|----------|
| 合成奖励 | +50万×5 + 15万×8 + 3万×15 = **715万** |
| 升级销毁 | -41.5万×20 - 3.6万×30 = **-908万** |
| **净销毁** | **-193万 BOX** |

## 6️⃣ 风险控制

| 风险 | 防范措施 |
|------|---------|
| 工作室批量注册 | 设备指纹 + IP限制（1IP最多3钱包） |
| 脚本自动开盒 | 行为检测 + 验证码 + 请求频率限制 |
| 多账号 | 社交绑定（Twitter/Telegram） |
| 批量升级 | 冷却期（升级后24小时不能交易） |
| 倒卖账号 | 首次升级需持有7天 |

### 检测机制
```javascript
const detectBot = (user) => {
  if (user.clickInterval < 500ms) return 'bot';
  if (user.ipCount > 3) return 'multi-account';
  if (user.openTimePattern === 'fixed') return 'script';
  if (user.deviceFingerprint === 'clone') return 'bot';
  return 'normal';
};
```
