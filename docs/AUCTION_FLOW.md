| 代币 | 价格 |
|------|------|
| 1 SUI | = 100 BOX |

| 销售价格 | BOX支付 | SUI支付 |

## 🔄 交易流程

### 1. 支付金额计算

```javascript
// 汇率
const SUI_TO_BOX_RATE = 100;

// 计算实际支付金额
function calculatePayment(priceInBOX, payToken) {
  if (payToken === 'BOX') {
    return {
      amount: priceInBOX,
      token: 'BOX',
      display: `${priceInBOX} BOX`
    };
  } else {
    const suiAmount = Math.ceil(priceInBOX / SUI_TO_BOX_RATE);
    return {
      amount: suiAmount,
      token: 'SUI',
      display: `${suiAmount} SUI (~${priceInBOX} BOX)`
    };
  }
}
```

## 📊 页面设计

### 1. 我的NFT页面
- 显示玩家持有的NFT
- 可以选择挂售

### 2. 拍卖/市场页面
- 浏览可购买的NFT
- 支持筛选（稀有度、价格）
- 支持排序（最新、最热、价格）

### 3. 挂售流程
- 选择NFT
- 设置价格（BOX或SUI）
- 选择拍卖类型（竞价/一口价）
- 确认上架

### 4. 购买流程
- 选择NFT
- 选择支付方式（BOX/SUI）
- 查看换算价格
- 确认购买

---

## ⚠️ 注意事项

1. **价格一致性** - 无论用BOX还是SUI，最终价值相当
2. **实时换算** - 支付时按汇率计算
3. **手续费扣除** - 卖家到手 = 售价 × 95%
4. **NFT转移** - 购买后NFT直接转入买家钱包
