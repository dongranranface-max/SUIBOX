## 2️⃣ 核心功能

### 2.1 邀请系统

**邀请奖励规则**

| 任务 | 被邀请人奖励 | 邀请人奖励 |
|------|-------------|-----------|
| 1位好友开盒 | +1次开盒 | +0.5 BOX |
| 3位好友开盒 | +2次开盒 | +1.5 BOX |
| 10位好友开盒 | +3次开盒 | +5 BOX |

**邀请条件**
- 持有10 SBOX才能邀请好友
- 每个账号只能获得一次开盒机会
- 邀请人可持续获得推荐奖励

**邀请链接格式**
```
https://suibox.io/r/{inviteCode}
```

### 2.2 开盲盒

**每日次数**

| 来源 | 次数 |
|------|------|
| 每日免费 | 1次 |
| 邀请奖励 | 最高6次 |
| 每日上限 | 7次/天 |

**开盲盒概率**

| 结果 | 概率 | 获得碎片 |
|------|------|---------|
| 普通碎片 | 50% | 1-2个 |
| 稀有碎片 | 15% | 1个 |
| 史诗碎片 | 3% | 1个 |
| 感谢参与 | 32% | 0 |

**保底机制**

| 保底次数 | 必得奖励 |
|---------|---------|
| 3次感谢 | 1个普通碎片 |
| 7次感谢 | 1个稀有碎片 |
| 25次感谢 | 1个史诗碎片 |

### 2.3 碎片合成

**合成配方**

| 消耗碎片 | 产出NFT | 奖励BOX | 预计开盒次数 |
|---------|--------|---------|--------------|
| 6普通碎片 | 1普通NFT | +5 BOX | ~12次 |
| 8稀有碎片 | 1稀有NFT | +8 BOX | ~54次 |
| 10史诗碎片 | 1史诗NFT | +15 BOX | ~334次 |

### 2.4 NFT升级

**升级配方**

| 消耗 | 产出 | 消耗BOX | 奖励BOX | 净消耗 |
|------|------|---------|---------|--------|
| 5普通NFT + 30 BOX | 1稀有NFT | -30 | +10 | -20 BOX |
| 4稀有NFT + 50 BOX | 1史诗NFT | -50 | +20 | -30 BOX |

**升级冷却**
- 首次升级后需持有7天
- 升级后24小时不能交易

### 2.5 NFT发行量

| NFT类型 | 发行量 | 定位 |
|---------|--------|------|
| 普通NFT | 100万 | 普及 |
| 稀有NFT | 10万 | 稀缺 |
| 史诗NFT | 2万 | 稀有 |

### 2.6 Staking

**持有NFT产出BOX**

| 持有NFT | 每日产出 |
|---------|---------|
| 普通NFT | +0.1 BOX/天 |
| 稀有NFT | +0.5 BOX/天 |
| 史诗NFT | +2 BOX/天 |

### 2.7 交易大厅

**功能**
- NFT挂单出售
- 购买其他玩家NFT
- 定价/竞价模式

**手续费**
- NFT交易手续费: 2%
- BOX交易手续费: 0.5%

## 3️⃣ 经济模型

### 3.1 代币经济

| 代币 | 用途 |
|------|------|
| SBOX | 平台治理代币 |
| 碎片 | 合成NFT原材料 |
| NFT | 收藏+Staking |

### 3.2 经济循环

```
                   ┌─────────────────────────────────────┐
                   │           交易所购买BOX              │
                   └──────────────┬──────────────────────┘
                                  ↓
┌──────────→ 开盲盒 ─→ 碎片 ─→ 合成 ─→ NFT ─→ 升级 ─→ 销毁BOX
│     ↑                                            │
│     │                       ↓                    │
│     └──────────────── Staking ── 产出BOX ─────────┘
│
邀请 ────────────────────────────────────────────────
```

### 3.3 通缩机制

| 行为 | BOX变化 |
|------|---------|
| 碎片合成 | +BOX（吸引玩家） |
| NFT升级 | -BOX（净销毁） |
| Staking | +BOX（持有收益） |
| 交易手续费 | -BOX（销毁） |

### 3.4 收入来源

| 来源 | 比例 | 用途 |
|------|------|------|
| NFT交易手续费 | 2% | 运营收入 |
| BOX交易手续费 | 0.5% | 回购销毁 |
| 提现手续费 | 1% | 支付通道 |
| 高级功能 | - | 定制NFT |

## 4️⃣ 用户流程

### 4.1 新用户流程

```
1. 访问网站 → 连接钱包
2. 首次免费开盒 → 获得碎片
3. 邀请好友 → 获得更多开盒次数
4. 集齐碎片 → 合成NFT
5. 持有NFT → Staking产出BOX
6. 升级NFT → 获得更高稀有度
7. 交易NFT → 获利
```

### 4.2 用户成长路径

| 阶段 | 目标 | 产出 |
|------|------|------|
| 新手 | 开盒集碎片 | 碎片 |
| 进阶 | 合成普通NFT | 普通NFT |
| 熟练 | 合成稀有NFT | 稀有NFT |
| 高手 | 合成史诗NFT | 史诗NFT |
| 商人 | 交易NFT | 差价收益 |

### 5.3 合约设计

```
┌─────────────────────────────────────────────┐
│              SUIBOX Contracts               │
├─────────────────────────────────────────────┤
│ 1. BoxCore (核心合约)                        │
│    - openBox() 开盲盒                       │
│    - getFragment() 获取碎片                 │
│    - dailyLimit() 每日限制                  │
├─────────────────────────────────────────────┤
│ 2. Craft (合成合约)                          │
│    - synthesize() 碎片合成NFT              │
│    - upgrade() NFT升级                      │
│    - burn() 销毁                            │
├─────────────────────────────────────────────┤
│ 3. Staking (质押合约)                        │
│    - stake() 质押NFT                        │
│    - claim() 领取奖励                       │
│    - unstake() 解除质押                     │
├─────────────────────────────────────────────┤
│ 4. Token (代币合约)                         │
│    - mint() 铸造                            │
│    - transfer() 转账                         │
│    - burn() 销毁                            │
└─────────────────────────────────────────────┘
```

## 6️⃣ 风控机制

### 6.1 防工作室/脚本

| 风险 | 防范措施 |
|------|---------|
| 批量注册 | 设备指纹 + IP限制(1IP=3钱包) |
| 脚本开盒 | 行为检测 + 验证码 + 频率限制 |
| 多账号 | 社交绑定(Twitter/Telegram) |
| 批量升级 | 冷却期(24h不能交易) |
| 倒卖账号 | 首次升级需持有7天 |

### 6.2 防女巫攻击

```
1. 钱包绑定设备/IP
2. 社交账号验证
3. IP地址限制(1IP=3领取)
4. 设备指纹(1设备=5钱包)
5. 行为分析(异常标记)
6. 持有时间要求(快照前30天)
```

### 6.3 安全审计
