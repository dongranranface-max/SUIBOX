## 🎯 项目概述

### 1.1 项目简介

SUIBOX 是SUI链上首个**NFT盲盒+DeFi**平台，结合了：
- 🎰 盲盒抽取机制
- 🧩 碎片合成NFT
- 📈 NFT升级系统
- 💰 DeFi质押产出

### 1.2 核心功能

| 功能 | 说明 |
|------|------|
| 开盲盒 | 每日免费1次+邀请获得 |
| 碎片合成 | 6/8/10个碎片合成NFT |
| NFT升级 | 低级NFT+BOX升级高级 |
| Staking | 持有NFT产出BOX |

### 1.3 解决的问题

1. **SUI链NFT应用匮乏** - 当前SUI链缺少有趣的NFT DApp
2. **用户获取困难** - 通过盲盒+社交裂变降低门槛
3. **经济模型创新** - 合成产出BOX，升级销毁BOX，实现通缩

## 💰 经济模型

### 4.1 代币用途

| 代币 | 用途 |
|------|------|
| SBOX | 平台代币，参与治理 |
| NFT | 收藏+Staking产出BOX |

### 4.2 经济循环

```
邀请 → 开盲盒 → 碎片 → 合成NFT → 升级（销毁BOX）
                                    ↓
                              Staking产出BOX
```

### 4.3 通缩机制

- NFT升级净消耗BOX
- 交易手续费销毁
- 回购销毁机制
