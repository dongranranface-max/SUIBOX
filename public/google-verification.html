To verify your site in Google Search Console:

  1. Go to Google Search Console
  2. Add property: suibox.site
  3. Choose verification method: HTML tag
  4. Copy the verification code below
  5. Update public/google Verification.html with your code

Example verification code:

<meta name="google-site-verification" content="abc123xyz789" />

Or use DNS verification: 

Type: TXT
Host: @
Value: google-site-verification=abc123xyz789
